# Indian Village Animation 🏡

A realistic Indian village animation project built using HTML, CSS, and JavaScript. This project showcases a lively countryside environment with moving clouds, flowing river animation, swaying mango trees, animated villagers, birds flying in the sky, and natural village scenery inspired by rural India.

The project features smooth character animations, realistic village elements, interactive mango effects, and beautiful nature-inspired visuals designed for frontend creativity and GitHub portfolio projects.

## Features

- Realistic Indian village scene
- Animated villagers and animals
- Mango tree movement animation
- Flowing river effect
- Flying birds animation
- Moving clouds and wind effects
- Interactive mango animation
- Smooth CSS animations
- Nature-inspired environment

Built using HTML, CSS & JavaScript.
